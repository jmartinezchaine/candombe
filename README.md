# Candombe App 🥁

Aplicación móvil desarrollada en **Flutter** y **Dart** diseñada para dar soporte y herramientas útiles durante los ensayos de Candombe.

## 🚀 Estado del Proyecto
* **Configuración Inicial**: Proyecto Flutter inicializado con soporte exclusivo para plataformas móviles (Android e iOS).
* **Entorno**: Flutter 3.35.6 / Dart 3.9.2.
* **Próximo Paso**: En espera de los lineamientos y requerimientos detallados del usuario para comenzar con el diseño y la implementación de las funcionalidades.

## 🛠️ Stack Tecnológico
* **Framework**: Flutter
* **Lenguaje**: Dart
* **Target**: Android & iOS (Mobile)

## 📂 Estructura del Proyecto (Planificada)
```text
lib/
├── main.dart             # Punto de entrada de la aplicación
├── core/                 # Utilidades, constantes, temas visuales y componentes compartidos
└── features/             # Módulos o funcionalidades de la app (e.g. metrónomo, afinador, etc.)
```

## 💻 Configuración Local

### Requisitos
* Flutter SDK instalado (versión 3.35.x o superior)
* Dart SDK (versión 3.9.x o superior)
* Dispositivo móvil/emulador configurado

### Ejecución
1. Descargar las dependencias:
   ```bash
   flutter pub get
   ```
2. Ejecutar el proyecto:
   ```bash
   flutter run
   ```
